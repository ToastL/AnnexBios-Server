Start the server

```bash
~ npm install

~ npm run build

~ npm run preview
```

info about the server

```
/api/movies                     # get the current running movies
/api/movies/MovieName           # replace "MovieName" to get information about a certain movie
/api/movies/MovieName/Property  # replace "Property" to get a specific property about the movie, for example description
/api/actors                     # to get current actors
```
